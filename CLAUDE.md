# CLAUDE.md

## Regras operacionais para agentes de AI neste repositorio

1. Sempre criar um plano de acao antes de executar mudancas.
2. Sempre fazer uma revisao tecnica do plano antes de implementar.
3. Sempre consultar documentacao oficial das ferramentas usadas.
4. Priorizar `ffmpeg` para tarefas de edicao/transcodificacao de video.
5. Ao finalizar uma tarefa:
   - executar 3 ciclos de verificacao (bugs, falhas, issues criticos);
   - gerar um mini relatorio de verificacao;
   - fazer commit e push no repositorio.
6. Sempre que possivel, usar ferramentas de execucao em background para agilizar o desenvolvimento.
